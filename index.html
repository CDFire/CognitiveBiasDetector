Cognitive Bias Detector

An AI-powered tool to identify and understand potential cognitive biases in text using a fine-tuned BERT model and XAI explanations.

View on GitHub

About The Project

This project leverages Natural Language Processing (NLP) and Explainable AI (XAI) to help users become more aware of common cognitive biases such as Confirmation Bias, Anchoring Bias, Availability Heuristic, and Sunk Cost Fallacy. By inputting text, the tool uses a fine-tuned BERT model to predict potential biases and highlights the words that most influenced this prediction.

The goal is not to definitively label text as "biased," but to provide an assistive tool for reflection and improved decision-making by understanding how language might reflect underlying cognitive patterns.

Key Features

  • Detects Confirmation Bias, Anchoring Bias, Availability Heuristic, and Sunk Cost Fallacy.
  • Utilizes a fine-tuned bert-base-uncased model.
  • Provides word-level explanations (XAI) by highlighting influential text.
  • Offers definitions and de-biasing recommendations for identified biases.
  • Simple and intuitive web interface built with Flask (for local execution).

Technology Stack

Python, Flask, Hugging Face Transformers (PyTorch), scikit-learn, transformers-interpret, HTML, CSS.

Get It Running Locally

To run this project on your own machine and interact with the Flask application:

  1. Clone the repository:
    git clone https://github.com//.git
    cd
  2. Set up a Python virtual environment and install dependencies:
    python3 -m venv venv
    source venv/bin/activate # macOS/Linux
    pip install -r requirements.txt
  3. Generate synthetic training data:
    python synthetic_data_generator.py
  4. Train the BERT model (can take a long time, GPU recommended):
    python train_transformer.py
  5. Run the Flask application:
    python app.py
  6. Open your browser and go to http://127.0.0.1:5000/.

For detailed instructions, please refer to the README.md file in the repository.
